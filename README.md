# QSCL Landing Page

A modern, dark-meets-light SaaS marketing website for QSCL (by Namo Labs), a quantum-safe cybersecurity/communication platform.

## Features
- Deep navy, indigo/blue gradients, soft lavender backgrounds
- Glassmorphism & Bento grids
- Responsive design

## Structure
- `index.html`: The semantic structure of the website.
- `styles.css`: Custom CSS variables and layout styling (Vanilla CSS).
- `script.js`: Sticky navigation and code tab interactions.
